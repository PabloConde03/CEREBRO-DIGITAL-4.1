# Convenciones de Etiquetas — Cerebro Predeterminado

> Taxonomía estándar y obligatoria para todo el conocimiento extraído.
> No usar etiquetas fuera de esta jerarquía sin autorización explícita del usuario.

---

## Estado de Flujo
Indica la madurez de una nota o proyecto.

| Etiqueta | Significado |
|---|---|
| `#estado/borrador` | Nota recién creada, sin revisar |
| `#estado/proceso` | En desarrollo o revisión activa |
| `#estado/completado` | Contenido terminado y verificado |
| `#estado/archivado` | Nota inactiva, preservada como referencia |

---

## Tipología de Nota
Clasifica la naturaleza del documento.

| Etiqueta | Significado |
|---|---|
| `#tipo/recurso` | Nota de conocimiento, snippet o teoría |
| `#tipo/moc` | Mapa de Contenido (índice temático) |
| `#tipo/proyecto` | Nota de proyecto con inicio y fin definidos |
| `#tipo/log` | Registro diario, bitácora o tracker |

---

## Contexto de Tareas
Clasifica las tareas por ámbito de vida.

| Etiqueta | Significado |
|---|---|
| `#tarea/personal` | Ámbito personal, salud, hobbies |
| `#tarea/profesional` | Trabajo, clientes, academia |
| `#tarea/mantenimiento` | Hogar, administración, gestiones |
